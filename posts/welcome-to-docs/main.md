# A New Era of Documentation

Welcome to the new **CtrlZeroDev Documentation Hub**. 

This platform is not just a wiki; it's a living knowledge base styled with our signature **"Liquid Glass"** aesthetic and powered by the **IMRF Engine**.

## How it works

We've adopted a "Post Bundle" structure inspired by the best in the industry (like Hugging Face). 

1.  **GitHub Powered** 🐙: All content lives in our [GitHub Repository](https://github.com/CTRLZer0/controldocs).
2.  **Markdown First** 📝: We write in pure Markdown with Frontmatter for metadata.
3.  **Asset Bundling** 📦: Images and assets live right next to the content in their own folder.

## Features

*   **Dynamic Rendering**: Powered by our custom IMRF renderer.
*   **Rich Metadata**: Authors, dates, and tags are first-class citizens.
*   **Automated Routing**: Just create a folder in `posts/` and it appears here.

## Contributing

Want to write a post?

1.  Clone the `controldocs` repo.
2.  Create a folder in `posts/my-topic`.
3.  Add a `main.md` with frontmatter.
4.  Open a Pull Request!

```yaml
---
title: "My Awesome Topic"
date: "2025-12-22"
tags: ["Tutorial"]
---
```
